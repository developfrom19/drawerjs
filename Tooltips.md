#### Describes customization properties for tooltips. 

***
### Configuration
```
tooltipCss: {
    color: 'white',
    background: 'black'
}
```
* Optional setting. If not specified, the settings shown in the example will be used.
* `color`:
    * Specifies the tooltips' font color.
* `background`:
    * Specifies the tooltips' background.
